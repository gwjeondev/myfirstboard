## 나의 첫번째 게시판 만들기


![home](https://raw.githubusercontent.com/won-developer/myfirstboard/master/images/home.png)

웹사이트 : https://my-first-board.herokuapp.com/


## 프로젝트 소개

네이버 디자인을 살짝 모방한 게시판 프로젝트 입니다.

__개발기간: 2020-02-28 ~ 2020-03-06__

## 기술 스택
- 백엔드: Node JS Express
- 프론트엔드: Vanila JS
- 데이터베이스: Mongo DB
- 인증: Passport JS
- 템플릿: Pug
- 스타일: SCSS


## 구현 기능
- 로컬 로그인
- 검색
- 게시글 업로드
- 게시글 페이징
- 게시글 조회수 및 좋아요
- 댓글 및 답글
